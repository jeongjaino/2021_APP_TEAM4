## 주요 기능 
- 투두 작성 : 난이도 설정, 시간 및 날짜 선택 가능, CRUD
- 이전 작성한 투두 조회
- 캐릭터 키우기 : 투두 성공시 레벨 증가, 터치 게임 및 흔들기 게임으로 골드 획득 시스템
